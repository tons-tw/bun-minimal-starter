# @tons-tw/tons

To install dependencies:

```bash
bun install
```

To start:

```bash
bun app-start
```

To dev:

```bash
bun app-dev
```

To format:

```bash
bun tool-fmt
```

To lint:

```bash
bun tool-lint
```

This project was created using `bun init` in bun v1.0.25. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
